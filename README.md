# 民宿酒店预订管理系统 python612

## 项目概述

本项目是基于 Python 开发的民宿酒店预订管理系统，旨在为用户提供便捷的在线预订服务，同时为酒店管理者提供一个高效的管理平台。

## 技术栈

- 后端：Python，使用 Django 框架
- 数据库：PostgreSQL
- 前端：HTML，CSS，JavaScript，Vue.js

## 功能模块

### 用户端

- 用户注册与登录
- 酒店搜索与筛选
- 酒店详情查看
- 在线预订与支付（*支付功能说明已被移除*）
- 订单管理与查询

### 管理端

- 房间信息管理
- 订单处理
- 用户管理
- 数据统计与分析

## 系统角色

- 普通用户：可注册、登录、预订房间
- 管理员：负责酒店房间信息管理、订单处理及系统维护

## 运行环境

- 操作系统：Linux 或 macOS（推荐），Windows 兼容
- Python 版本：3.6 或以上
- PostgreSQL 版本：9.5 或以上

## 部署步骤

1. 环境准备：安装 Python、PostgreSQL 和其他依赖库
2. 数据库配置：创建数据库及用户，配置权限
3. 程序部署：克隆代码仓库，安装依赖，迁移数据库，运行应用

- 基于成熟框架开发，系统稳定可靠
- 前后端分离，便于维护与扩展
